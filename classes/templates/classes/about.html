Our Week-by-Week Progress

  • Week of March 26th

    Created Python script to scrape building/class data to put into a Postgres database, used Leaflet/JS to build a map with basic features, began implementing searching.

  • Week of April 2nd

    Started hosting the site on Heroku, linked the frontend map design and the backend database using the Django framework and implemented a search bar feature to query the database and maps a result to the map.

  • Week of April 9th

    Restructured search logic to improve searching, refined searching by adding filters by day/time, added resource overlays to the map

  • Week of April 16th

    Fixed bugs regarding search filters, set up CAS authentication, added function to save/favorite classes, and improved the UI substantially (fullscreen map, overlays).

  • Week of April 23rd

    Added overlay to display saved classes, implemented a function to search by clicking on a building, added additional buttons to the map, and eliminated pesky bugs.

  • Week of April 30th

    Implemented a heatmap to indicate the number of students in each building, refined search results page to display as an overlay, substantially improved UI based on user feedback
